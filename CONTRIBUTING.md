# Contribuindo para o MrCayres Association

Ficamos muito felizes pelo seu interesse em ajudar! Para manter a organização, siga estes passos:

## Como contribuir?
1. Faça um **Fork** do projeto.
2. Crie uma **Branch** para sua funcionalidade: `git checkout -b feature/nova-funcionalidade`.
3. Faça o **Commit** das suas alterações: `git commit -m 'Adiciona nova funcionalidade'`.
4. Faça o **Push** para a branch: `git push origin feature/nova-funcionalidade`.
5. Abra um **Pull Request**.

## Padrões de Código
* Mantenha o código limpo e comentado.
* Siga a indentação de 2 espaços.
* Documente novas funcionalidades no README.
